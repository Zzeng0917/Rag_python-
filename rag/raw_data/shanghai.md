# 上海速览 Introducing SHANGHAI

上海一直是受外来文化影响的一座城市，从中国最早开埠的城市之一到繁华的“十里洋场”，各国文化自此融会贯通，外滩老式的西洋建筑与浦东现代的摩天大厦交相辉映；徐家汇大教堂圣诗声声，静安寺香烟袅袅；过街楼下的麻将老人，弄堂里的足球少年；群众小剧场的沪剧、滑稽戏，大剧院的交响乐、芭蕾舞；老饭店的本帮佳肴，杏花楼的广式粤茶，红房子的法国大菜；上海老街的茶馆，衡山路的酒吧，中西合璧，各有各精彩。

南京路和外滩：上海和上海商业的标志，白天热闹，晚上更热闹  
人民广场：上海的绝对市中心，从这里可以方便通达到上海的任何景点。  
淮海路和徐家汇：前者是上海公认的最美丽、最摩登、最有“腔调”和情调的一条街。后者则兼容并包，还有些像北京的中关村。  
浦东新区：拔地而起的新区，代表着最现代最前卫的上海。

不系舟靠岸比萨港 这是一座开放而又保守的城市。在雕刻着沪江历史的石库门中穿行，即能听到吴侬细语声声，也能耳闻洋话大论篇篇。就像一杯花雕兑上玛格丽特的混搭韵味，又似乎如一位穿着孔雀蓝绣大红牡丹旗袍的混血美女款款走来。

# 特别推荐 Special

# 上海弄堂

北京的路横平竖直，要说往哪儿，都说往北，往东，透着大方的气派；上海不一样，说不定哪一处一拐，又是哪一处一曲，上海的弄堂尤其如此，就像上海人的肠肚，曲里拐弯，含蓄得很，上海的故事 , 大概就是上海弄堂的故事吧。

# 本帮菜

本帮菜的特色是既有甜淡的清淡素雅，也有“浓油酱赤”的油重味浓特色。如狮子头、红烧肉、油面筋、咸肉老汤百叶。相对于本帮菜，更受人青睐的是上海的小吃，汤包、百叶、油面筋被认为是不得不尝的 “三主件”，而更加让人如雷贯耳的是上海的生煎包。

# 海派清口

周立波创立的海派清口，如今在舆论和人气上，已近与郭德纲代表的相声和赵本山代表的二人转、喜剧鼎足而立。海派清口脱胎于“滑稽戏”，海派代表上海，清代表至尊，至诚，至真，口代表嘴。是一种类似于演说的单人表演形式。他用一种幽默的方式解说社会热点、焦点，单纯地表达一种快乐的生活方式。

最佳旅行时间 上海作为世界大都市来说，其实并没有严格的季节限制，但是每年春天（3-5 月）上海青浦梅花、奉贤油菜花、浦东桃花、松江的兰笋次第开放，煞是好看。而秋天（9-11 月）是品尝大闸蟹的最佳时节，所谓“秋风响，蟹脚痒”，此时的大闸蟹蟹黄足、蟹脂厚、蟹肉美，令人垂涎欲滴。

# 亮点 Highlights

# $\star$ 外滩的一早一晚最精彩

除了各种风格的建筑，早上看晨练，晚上看夜景，才是外滩（见第5页）的精华，6-7 点钟过去，可以看到有人打太极、有人舞剑。整个上海也仿佛刚刚醒来。外滩的夜景，交织着现代和过去。夜幕降临，灯火辉煌、黄埔江上游轮偶尔的鸣笛，可以说是每个人的上海初印象。虽然俗了点，但是也只有自己亲身感受才过才称得上来过上海。

# $\star$ 梧桐 $^ +$ 老洋房 上海老街的别样风情

在温暖的午后，找一条幽静的上海老街，这儿梧桐满街、绿荫如盖，一幢幢老洋房点缀其间，在这个国际化的大都市，沉淀原始的上海味道。遛弯或者找一辆自行车，都足够惬意。

体验去处：上海的老洋房主要分布在四个市中心区：徐汇区、长宁区、黄浦区和静安区。汾阳路、愚园路、武康路、湖南路、思南路、长乐路、华山路、兴国路、茂名路是老洋房最集中的区域。马勒别墅和周公馆是其中的代表作。（行程推荐可参看第24页）

# $\star$ 石库门 $^ +$ 晾衣竿 最浓郁的上海市井缩影

在田子坊（见第 8 页）、上海新天地（见第 7 页），你经常可以看到从窗户里伸出的竹竿，晾着刚洗的衣服，伴着上海特有的石库门建筑，成为上海市井最具代表的一道风景线。

# $\star$ 除了小资，还是小资

上海对小资情结有一种近乎狂热的迷恋，大到大大小小的创意园区，小到街边的小店或酒吧，除了大家耳熟能详的衡山路（见第 9 页）、1933老场房（见第 9 页）和 M50 创意园区（见第 9 页）之外，上海还有更多的所在（见娱乐部分见第 24 页），上海的灯红酒绿，更多的是一种与生俱来的气质。

# 景点 Sights

到上海，你永远不会觉得无事可做，你可以到外滩尽情感受夜的风情，去徐汇区的天平路和湖南路找最漂亮的老洋房，在老上海的里弄感受历史的沉淀，或者索性也彻底小资一把，去田子坊、新天地静坐冥想，任时光流逝；衡山路酒吧街对饮狂欢；1933 老厂房这些艺术仓库看看中西混搭的艺术魅力。最后千万别忘了登上东方明珠，放眼看看这个国际化的大都市。

# $\circledast$ 滨江大道

滨江大道是位于黄浦江边的一条人行大道，与外滩隔河相望，华灯初上时，一侧是璀璨的东方明珠和现代化摩天大楼，另一侧是复古典雅的外滩万国建筑博览群。现在的滨江大道，俨然已经成为上海的“小外滩”。在滨江大道中央，有由 21 个喷水头组成的喷水广场，在滨江大道富都世界段，由旧码头改建的游艇码头独具情趣，码头上竖立着巨大的铁锚。巨大的喷水池中安装了彩灯，在夜色中大放光彩。令人流连忘返。

# 如何观赏滨江大道

# $\otimes$ 金茂大厦 88 层观光厅

# $\pmb { \mathfrak { O } }$ 外滩

有人说“外滩的故事就是上海的故事”， 外滩的精华就在于 52 幢风格各异被称为“万国建筑博览”的外滩建筑群。面积不算大的外滩集中了20 多幢不同时期、不同国家、不同风格的建筑。著名的中国银行大楼、和平饭店、海关大楼、汇丰银行大楼再现了昔日“远东华尔街”的风采，这些建筑虽不是出自同一位设计师，也并非建于同一时期，然而它们的建筑色调却基本统一，整体轮廓线处理也惊人地协调。无论是极目远眺或是徜徉其间，都能感受到一种刚健雄浑、雍容华贵的气势。

# $\pmb { \mathfrak { O } }$ 南京路

南京路，每个第一次到上海的人都会去逛一逛。市百一店（原“大新公司”）、永安商厦、上海时装公司（原“先施公司”）和第一食品商店（原“新新公司”）这“四大公司”，交叠着南京路的过去与今天。

从河南路口开始到步行街结束，继续向东，一直到南京东路外滩，行人似乎又坠入上个世纪三四十年代。：狭窄的街道，高大的建筑，灰色调的围墙和百叶窗，高高的天花板，尖尖的屋顶，每一栋楼都记载了一段曲折跌宕的故事。

南京路分为西路和东路，一般说的南京路步行街，指的是东路。

# 多伦路

# 东滩湿地

崇明东滩恰好位于亚太地区候鸟南北迁徙的必经路径上，是过境候鸟停歇、补充能量的驿站。每年在崇明东滩过境中辗转和迁徙越冬的水鸟数量可达数百万只。

离开城市的喧嚣，踏上软软的土地，心中已然惊叹不已。湿地上到处是水草和芦苇，湿气袅袅，好像地面上飘起了云雾。在这里，目之所及，只有蓝蓝的天，白白的云，红红的太阳，嫩嫩的草，宽宽的芦苇荡……除此之外，据说这里也是崇明岛日出最美的地方。

餐饮：东滩观鸟后可回到陈家镇，找一家朴素而洁净的饭馆。要上一盘大闸蟹，几样海鲜，大快朵颐一番。陈家镇是崇明最大蟹市，也是中国最大的蟹市，上海的蟹大部分来自陈家镇。这里稀有的海鲜较多，而且价格比别处都便宜。

看点 : 日出观鸟：到东海观鸟，最佳时机是早晨，因为东海日出是崇明的一大自然胜景，崇明东滩地处东海最前沿。早晨既能观鸟又能观日出，真是一举两得。

地址：崇明岛东端，东旺路，长江入海口处费用：周末80元，周一至周五45元开放时间：9:00 - 17:00

# 住宿 Sleep

作为大都市，上海有着各种级别的酒店和旅馆。上海的酒店和旅馆分布广泛，几乎涵盖了所有商圈、景点，甚至住宅区。如果主要的目标景点位置比较集中，在附近寻找合适的酒店是最方便的。

位于交通枢纽附近的酒店房价可能稍贵，但可以省下不少交通费和宝贵的时间。其中黄埔区、静安区、南市区和虹口区是上海传统商业中心，浦东新区、长宁区和徐汇区是目前新兴的商业圈。

上海的四五星级、甚至超五星级的高级酒店数不胜数，多数集中在上海最繁华的商业区，并形成了静安、虹桥、浦东三大宾馆群。高级酒店的价格在六百至数千元不等，这类酒店的管理水平和服务质量均在国内保持着领先地位，有的堪称世界一流，是商务旅客的首选。如上海外滩中心威斯汀大饭店等。

各种经济型连锁酒店在上海分布非常广泛，几乎能在任何地段找到它们。其中星程、如家、汉庭、莫泰、锦江之星、7 天等都是知名品牌。连锁酒店价格适中，一般在一百至三百多元不等，客房条件虽比不上高级酒店，但比同等价位的其他旅馆要舒适和安全。

对年轻的自助游背包客来说，价格是最重要的考虑因素，青年旅馆、求职旅馆、高校招待所和一些日租房都是不错的选择。床位大概在 40-60元之间，标间大概在200-300之间，大部分此类旅馆交通出行还算方便。

上海的酒店，价钱贵在地段，即使是普通的经济型酒店，在像外滩这样的黄金地段，房价也可能高达 300 元，此外，周末和黄金周价格都会上调，最好提早预订。

# 购物 Shopping

上海汇集了世界和全国各地名品，在上海购物，地铁是既方便快捷又能一网打尽的交通方式：从徐家汇到淮海中路是购物的首选线路。尤其是淮海中路。

2号线的南京东路站、南京西路站（要点是梅龙镇、中信泰富和恒隆广场）、静安寺站串起了南京路的过去与现在。每逢换季、节庆或店庆时，各大商场都有惊人的折扣，是购物的好时机。

此外，像进京乳腐、枫泾丁蹄、五香豆（郭记兴隆五香豆店）、上海织绣、嘉定黄草编、南汇水蜜桃、金山农民画、嘉定竹刻、上海雕刻、城隍庙梨膏糖、南翔小笼包、上海邵万生黄泥螺等上海特产也很受欢迎。

商场营业时间一般在 9:00-21:00 或 10:00-22:00，周末和节假日适当延长，个别例外。

● 商场前有人发给小商品，不必接手，以免纠缠。

● 换季、节庆、店庆是商场打折的时候，别空手而归，以免后悔。

在上海部分商厦内实行“三色”标价签制度:蓝色— 明码标签；黄色——削价标签；红色——议价标签。

地处黄浦区浏河路、东台路口，是一条在海内外具有较高知名度的旧工艺品特色街，被称为上海的“琉璃厂”。以经营旧工艺品为主，包括瓷器、玉器、铜器、木器、字画、文房四宝、杂件等。这里的每个店家的店堂装饰都自成风格，极具特色。  
到达交通：公交17、864路东台路站下

# 浦东机场免税店

只对出境旅客开放，因此只有飞国际航班的时候才能看到免税店。免税店集中于国际出发区 15-20 号登机口附近。日上免税行经营世界畅销香烟、酒品、香水及各式化妆品。东权免税店主营箱包、皮具、名表、服饰等百货类商品以及各类免税食品。  
到达交通：浦东机场专线车和磁悬浮列车

# 娱乐 Entertainment

上海是名符其实的“娱乐之都”，甚至用纸醉金迷来形容上海也不为过，从酒吧、各种电影节、音乐会，到各类体育赛事（如 F1 赛车），再到五花八门的主题展馆和乐园，你可以在这里彻夜狂欢，或者只是安静地看一场歌舞表演，选择永远比决定要多。

# 校园酒吧

主要位于城东北的五角场，邻近复旦、同济等高校。这类酒吧最大的特色就是前卫，前卫的布置、前卫的音乐、前卫的话题。变异夸张的墙面画，别出心裁的题记，大多出于顾客随心所欲的涂写，不放流行音乐，没有轻柔的音乐，从头到尾播的都是摇滚音乐。每逢周末有表演，常有外国留学生夹杂其中，裸着上身忘情敲打。

代表：HardRock（硬石）（政通路 315 号）、黑匣子（四平路 2069号近国定路 ) 等

# 音乐酒吧

主要讲究气氛情调和音乐效果，配有专业级音响设备和最新潮的音乐CD，时常还有乐队表演。柔和的灯光、柔软的墙饰，加上柔美的音乐，吸引着不少注重品位的音乐爱好者。日常经营往往都有音乐专业人士在背后指点，有的经营者就是音乐界人士和电视台、电台音乐节目主持人。代表：CJW 爵士酒吧（延安东路 222 号外滩中心 50 楼近江西中路）、Cotton Club（棉花俱乐部）（淮海中路1428号近复兴中路口）等

# 商业酒吧

追求的是西方酒吧温馨、随意和尽情的气氛，主要集中在大宾馆和商业街市。高档商业酒吧如外滩三号、九重天（金茂 87 层，世界最高酒吧）的最低消费都不菲，至少数百元。新天地、衡山路、茂名南路、淮海路、复兴公园是上海商业酒吧最集中的几个区域。

代表：M-BOX（淮海中路 1325 号）、东魅（太仓路 181 弄新天地北里17号楼）等

# 剧院、演出和比赛

上海剧院多，演出也多，有歌剧、话剧、戏曲、音乐会，以及大大小小国内外的明星演唱会。各种各样的演出体现海派文化超强的包容和吸纳能力。此外，上海有多家设施豪华的高档影院，虽然平日放映的影片与全国各地并无太大的不同且票价昂贵（夜场通常需要 60-80 元），但仍有值得关注的热点。每周二全上海电影院全天半价，周六周日上午半价。

很多电影会将首映式放在上海，同时开影迷见面会、签售等一系列宣传活动。各大影院也会售卖当季电影的周边产品如玩具、书籍、服装等。每年上海国际电影节期间会有世界各地优秀影片的展映，如果在这段时间前往上海，一定不要错过各大影院的信息。

上海也是国际体育赛事集中举办地之一，包括 NBA 篮球赛上海站、F1国际赛车大奖赛和国际田径黄金联赛等都会在上海举行。

# 茶坊和评弹书场

不同于中国传统的茶馆，茶坊一般总是装修得较为现代而温馨，更符合年轻人的口味。尤其是餐桌上的蜡烛和温柔的音乐，以及浪漫的秋千椅，十分有情调。茶坊的消费，比之酒吧、西餐厅要便宜得多，而且价格统一。饮料的种类有最经典的泡沫红茶、各式口味的奶茶、可以无限续杯的茶水等。装的容器也不尽相同，还有各种餐饮提供，十分雅致有趣（推荐：百米桌游主题茶坊，黄浦区浙江中路 229 号百米香榭商业街 3 楼 310商铺，1/2/8/10 号线和人民广场公车可达，人均 35 元；天下一家茶坊，浦东新区花木樱花路435号,798、方川专线、975路可达，人均75元)。

苏州评弹在上海流传广泛，有百余年的历史。虽然现在评弹已经不是大都市的流行娱乐活动，但仍有味道，雅庐书场（顺昌路315号）就是其一，已经有 91 年历史了，不过人气大不如前。

# 漫步路线 1

# 中山公园—愚园路—江苏路—华山路—徐家汇

时间 :1 － 1.5 小时

推荐 : 愚园路 在 1918 年的上海版图上，这里的确是一个名为“愚园”的私家花园。经历过旧上海租界的喧嚣，住过七十二家房客，历经半个多世纪煤卫合用的热闹之后，现在的它，像老克勒一样平和温婉又不失风度。愚园路是典型的上海马路，曲曲折折从最西端一直延伸开来，弄堂的号码一直排列到1600多，每条弄堂里更有数不清的住家。

从西往东慢慢走来，两侧幽深的弄堂里，一幢幢惊艳的小楼，随时跃入视野。老房子，特别是有故事的老房子，总让人心生出许多莫名的联想。路两边，是生长了半个多世纪的梧桐，它见证了上海的变迁。

# 驻足点

汪公馆　地址：愚园路 1136 弄 31 号

60 多年前王伯群为娶大夏大学校花保志宁而建，因汪精卫住过而得名。如今再看这座哥特式城堡建筑，房子依旧是那幢房子，只不过门口挂上了长宁区少年宫的牌子。有时候白天走过，还能听见从窗户里飘出各种各样的器乐声，或悠扬，或急促。

愚园路 749 弄 这是一处从外面绝对发现不了的别墅群，最为出名的是 63 号、65 号、67 号。从 749 弄拐弯进去，整个建筑群的入口相当开阔，但至后来，越显幽秘，主人也越显尊贵。最后几扇小门，通往熙熙攘攘的闹市，据说也是为方便逃生。乱世里的房屋，留着乱世里才有的门。

# 徐家汇—衡山路—淮海路

时间 :1 小时

推荐：衡山路，走过才知道，白天的衡山路和晚上判若两人。不宽的马路，干净的街道，两边风格迥异的建筑，一切都在日光下简单直接，和暧昧无关。

最喜欢路两边浓密的梧桐。蔓延了好几个街区，悠远的历史使树的枝叶异常繁茂，经过修剪的树枝密密地遮盖了路的上空，整条路变成长长的拱形走廊。

衡山路上隐匿着很多历史久远的老公寓和私家洋房，它们如隐士般在岁月变迁中保持着自己独有的姿态。偶而开合的铁门，备显神秘，只留给路人匆匆一瞥。门内繁茂的绿树枝叶探出门外，攀上山墙的青藤，夕阳下鲜艳的建筑轮廓，路灯下建筑拉长的身影，总会令我们有一点心动。

# 驻足点

# 香樟花园

地址：衡山路2号甲(近桃江路)

衡山路上的香樟花园，是上海很有气质的咖啡馆。那棵高大的香樟树穿墙而出，树与建筑成了不可分割的整体，营造着独特的韵致。临街的一面，拥有明亮通透的开放式玻璃窗。

如果走累了，可以在这里选一个靠窗的座位，点一杯蓝山咖啡，把思绪抛到九霄云外，只管看着窗外发呆，让路人成为自己的风景，也让自己成为路人的风景。

# 小红楼餐厅

地址 : 衡山路 811 号

3层的红色小楼，充满了一种诱惑。踏入小红楼的那一刻，沉浸在那份优雅和怀旧里。据说，这里是百代电影公司在中国的总部的前身，1983 年，又成为中国唱片总公司上海公司所在地。

# 漫步路线3

# 延安西路—新华路—淮海路—华山路—徐家汇

时间：1.5 小时

# 推荐：新华路

走在新华路上，路旁的梧桐树枝叶茂盛，树冠在路中间合拢，阳光只能透过枝叶稀疏处洒下，在路面上形成斑驳的光点。走过上海民族乐团，隐隐约约还能听见庭院深处传来的悠悠琴声。

如今，临街的老建筑被整修一新，围墙上布置了各色浮雕。可惜的是，新添的几座现代主义雕塑，总让人觉得有些突兀。

# 驻足点

新华路200号 一座中国传统宫殿式建筑，重檐大屋顶，红色琉璃筒瓦，灰白墙壁，沉稳而庄重。

新华路231 号 这栋3 层高的英国乡村式花园住宅，白色的粉墙露出黑色的木框架，老虎窗透出一丝优雅。

新华路329弄32号乙 早年间的瑞典公使馆，匈牙利建筑师邬达克设计。  
建筑属英国民居形式，局部带罗马城堡式，在附近的洋房群里独具特色。  
据说解放前此建筑曾转让给盛宣怀家族，至今盛家尚有后裔居住在内。

# 漫步路线 4

# 漕宝路地铁站—漕宝路—桂林路—田林路—宜山路

时间 :2小时

# 推荐：田林路

从漕宝路地铁站走出来，沿着漕宝路一直朝桂林路走，便是飘香四溢的桂林公园，也许并不是所有人都知道，桂林公园原称“黄家花园”，是旧上海大亨黄金荣的私人花园别墅。

转个弯，走十几分钟就是繁华的田林路。下班的时候经过这里，除了两旁各式各样的小店，还有偌大的夜市。在路边水果店里称上几个橘子，来斤炒货店里的瓜子，边走边看边吃，还真能淘上几样家中缺少的小杂物。

# 漫步路线 5

# 徐家汇—建国西路—建国中路—建国东路—西藏南路

时间：2小时

推荐：建国西路 沿街徐行，老梧桐树叶子随风沙沙作响，周遭的喧闹顷刻间都不复存在，在浓郁的人文气息里自由地让思绪静静蔓延。

这条路上，可看的东西太多：太原路的别墅，天平路的建筑，从陕西南路拐个弯，是最具书香气质的绍兴路。虽然是一条很不起眼的小马路，从东到西，不过两三百米，不熟悉的人会将它错过。马路窄，两边是高大的梧桐树，梧桐树影洒下来，使这条街显得更加幽静。因为路上没有公交车，天气好的日子，便经常可以看见三三两两的人围在马路中间聊天，看样子像是这里的居民，这就有了很家常的市井味。

# 驻足点

# 太原别墅

地址：太原路160号  
建于 1928 年。抗日战争结束后这里曾作为美国总统特使马歇尔将军的寓所，故又以“马歇尔公馆”而闻名。  
建筑风格为法国晚期文艺复兴式，其最显法国特色的部分是它的孟莎式屋顶和法式老虎窗。

# 汉源书店

地址：绍兴路 27 号

书店不大，看上去很宽敞。犹如自家的书房，靠窗摆放着深绿色的沙发，沙发间的桌上有一盏别致的小灯，带着旧上海的味道，又渗入了西式浪漫风情。沙发旁有一排书架，陈列着各式书籍。各式古匾挂于四周壁墙，8mm 电影放映机，古朴的手摇电话，宁静而有序地铺陈在眼前。

# 漫步路线6

# 外滩—南京东路—南京西路—静安寺

时间：1 小时

# 推荐：南京西路

那日跟朋友一起走，他指着静安别墅说，这里在 1930 年代开有很多珠宝店、皮货铺，电影《色•戒》中那间珠宝店就在这里，而且也确有其处，据说是张爱玲的好友炎樱家开的。

当年的珠宝店早已无影无踪，只留下静安别墅，洗去了旧时的浮华，不再带有旧上海那股浓艳的胭脂粉味。其间，住着老上海的后代们，过着柴米油盐的平淡生活。

说到写尽上海金粉幽怨的张爱玲，自然要拐个弯去看看张爱玲故居，就在常德路和愚园路的交汇处。遥想当年，她也是常常行走在南京路的夜色中吧。

# 驻足点

# 张爱玲故居

地址:：常德路195号常德路 195 号，常德公寓，旧时唤作“爱丁堡公寓”，1939 年的张爱玲与母亲和姑姑就住在 51 室；1942 年搬进了 65 室 ( 现在为 60 室 )，直至1948年。在她的《公寓生活记趣》中，记录了这段生活。

# 漫步路线 7

# 南京西路—陕西南路—长乐路—镇宁路

时间：1.5小时

# 推荐：长乐路

长乐路以乌鲁木齐为界，划分为两个世界。往西是老建筑的天地，以民居为主，其中不乏历史保护建筑。安静地走走停停，在不期然中收获惊喜。这些还不是全部，穿梭在长乐路大大小小的弄堂中，那才是精髓所在。夜色渐临，老洋房庭院前，一老人躺在树荫下养神，摇椅旁的收音机，播着咿咿呀呀的昆曲。

往东则是有名的时尚一条街。服装店、饰品店一家挨着一家，吸引大小艺人名人出没。老外们会把店开在3层老洋房里，黑白色系，复古墙面，嘎吱作响的木头楼梯，时尚的衣饰，据说还可以度身定做。

如果从东往西走，富民路附近有不少沪上有名的小饭店，如保罗酒家、阿毛餐馆。

# 漫步路线 8

# 淮海中路—汾阳路—东平路—乌鲁木齐路

时间：40 分钟

# 推荐：东平路

东平路不长，因一部《大城小事》而蜚声海外。这部被戏称为“最佳上海风光片”的影片，就是在这里取景。

从汾阳路过来，沿着东平路慢慢走，不过 200 多米，没有公交车，行人也不多。惟其如此，才能一直用最闲适的心情漫步。直走，就是衡山路，再走，就是乌鲁木齐路，夜上海最繁华喧闹的场所。

可以避开繁华，只需要朝里面走上十几米，往回望，汽车还能看到，车声却奇迹般地消失了。走在其间，内心平静下来，脚步也不由自主地缓慢下来。

# 驻足点

# 宝来纳餐厅

地址：汾阳路 150 号

气势非凡的灰白色洋楼，具有法国文艺复兴时期的建筑风格，如今已是上海滩上一家知名的德国餐厅。在这里能尝到著名的德国烤猪肘，而它的自酿啤酒据说是全上海最好的。当年国民党高级将领白崇禧曾居住于此，之前则是法籍冒险家司比尔门的私家花园。

# 爱庐

地址:东平路9号

一幢法国式的花园住宅，为宋美龄的陪嫁。典型的法式花园洋房，墙面绕着随季节变幻色彩的藤蔓，外墙嵌着或黑或白或黄的各色鹅卵石，屋面为孟沙坡面式的，铺着红色的平板瓦，干净洗练而不落俗套。

# 漫步路线 9

# 淮海路—天平路—武康路—湖南路—兴国路

时间：70分钟

# 推荐：武康路

武康路很长，走这条路需要一定的脚力。但它的幽静，却让人无法放弃。没有到过武康路的人，很难体会到上海老马路的静谧和幽雅。一条不算狭窄的路，斜斜地延伸着，没有丝毫张扬的意味。马路两旁的老别墅在梧桐的绿荫里露出一角，不显山不露水，静静地享受着绿意和阳光。

上海不是一座传统的中国城市，它的文化是开放的，洋为中用，中西结合，武康路就是最好的表达，有众多的西式建筑，又沉静内敛。

从武康路往周围辐射，还有太多可以走的路、可以观赏的街景。天平路上的服饰小店让人心动，与淮海路交汇处的 Hands 值得一看。湖南路上住过赵丹、阮玲玉，这些风流人物为湖南路刻下了别样风情。

# 驻足点

# 武康路

地址：107弄2 号  
陈立夫、陈果夫在沪住宅，一幢英国乡村式花园别墅。这幢建于 1946年的3层花园洋房，造型十分典雅。目前，这幢住宅内居住着好几户人家，算是“四大家族”豪宅里最平民化的一幢了。

# HANDS

地址：天平路 31 号

一家很不错的服装外贸小店，品质很不错。店内还有小小的玩具屋，里 面摆满了各式各样从国内外搜罗来的奇怪玩意。

# 赵丹故居

地址：湖南路8号

深深的院墙，一扇花格的铁门，就如同一道阻隔在岁月之间的栅栏。靠近小洋楼的地方有一棵枝繁叶茂的大杉树，虚虚实实地护着小楼的入口处，这公寓也就愈发显得静谧了。赵丹的寓所在第3层。

# 漫步路线10

# 4 号线海伦路口—溧阳路—四川北路—多伦路—山阴路

时间：1 小时

# 线人推荐：多伦路

从 4 号线出来走 5 分钟就是幽静的溧阳路。从喧嚣的四平路弯进安详的溧阳路，喧闹一下子就被浓密高大的梧桐树荫赶走了。路旁尽是花园洋

房，车子不多，行人寥寥，两旁矮墙上还有风景画，恍如衡山路的安静版。  
沿着溧阳路一直走，就是多伦路。

夕阳洒在多伦路上，一条不足 600 米的街道，沿街的书贩和古董摊吸引路人驻足。一条曾经同时住过鲁迅、叶圣陶的街道，里弄深处的老建筑流转着1930年代风情，湿润的空气氤氲着怀旧的味道。

# 驻足点

# 老电影咖啡吧

地址：多伦路123号

上海首家主题电影吧，2楼可以看由老式电影机放映的上世纪二三十年代老电影。一杯咖啡，一盏充满怀旧情调的街灯，一只被岁月封尘的邮筒，就如同这条街上不变的老建筑。

# 孔公馆

地址：多伦路 250 号

孔祥熙、宋蔼龄夫妇在上海有4处豪宅，多伦路上的“孔公馆”是其中之一。漫步在多伦路上，向 250 号里张望，那气派的外墙，那华丽的拼花瓷砖，老建筑的伊斯兰风情扑面而来。据说，孔祥熙喜欢在周末坐夜班火车到上海府邸，设宴招待宾客。

# 方向标：奉贤·油菜花

# 青浦环湖赏梅必去三处

1、东方绿舟。嫁接在百年老梅残桩上的烂漫梅花，连成一片片香雪海，串成一条条梅花道，依水而伴，随风而舞，美不胜收。

2、大观园的红楼。大观园淀山湖畔的梅园玲珑隽秀。梅园别称梅坞春浓，是个 30 多年的老园子。是上海地区最大的赏梅胜地，也是“江南四大梅花园林”之一。园中红梅、粉梅、绿梅、白梅、腊梅争奇斗艳，“宫粉梅”以冷艳之美著称，而“小绿萼”则玲珑可人。如果来得巧，还能跟着刘姥姥一起去探梅。

3、朱家角。这都是在一条线上，自驾很方便。身处江南，寻梅未必需要踏雪。初春时分，随意徜徉于水乡古镇内的课植园（朱家角镇北首西井街），不经意间便能偶遇几株梅树，虽不成规模，却也开得有滋有味。

# F1 上海站赛事

F1 是 Formula One 的缩写，中文叫做一级方程式赛车。以共同的方程式（规则限制）所造出来的车就称为方程式赛车，而其中等级最高者就属 F1。F1 在现今赛车领域所代表的地位就有如“奥运会”或是“世界杯足球赛”。目前 F1 共有 10 支参赛车队，每场比赛最多只有 20 位车手上场，每年规划有 16 至 19 站的比赛，通常约在 3 月中开跑，10 月底结束赛季。每站比赛可吸引超过 10 亿人次通过电视转播或其他媒体观赏这世界顶级赛事。

举办时间：每年4月中旬举办地点：上海国际赛车场

# 公交车

# 地铁

上海地铁目前已开通 14 条，单次票按路段收费，3-7 元不等，相比之下，套票更划算。1 日票（18 元）和 3 日票（45 元），可以在 24 小时和 72 小时内无限次乘坐。地铁首班车一般为 5:30-6:00，末班车为22:00-22:30，对 3 元的低票价来说，3 日票基本上 3 天内（72 小时）使用 15 次就回本了。 如果频繁在上海乘坐交通工具，建议购买地铁一日通或者三日通票，即使不买，为了方便，公交卡也是必要的。具体请查询上海地铁官网：http://www.shmetro.com/

赵沫沫 上海的地铁和北京的还是不一样的，比较错综复杂，通常一条地铁线上会有4个以上的换乘站，而且票价分段，一般没个5元是下不来的，不过我们一到虹桥火车站，就在地铁售票口买了一卡通，这样穿梭在网状的上海地铁内就顺利多了。地铁站有方便的咨询台，免费获取地铁图。

ViannYeah 到上海旅游几天的朋友，最好先到地铁站买张上海交通卡。不仅不用劳心找零钱坐车坐地铁，而且半小时内换乘还有优惠。例如，半小时内地铁换乘公交车，2 元的公交车车费减1 元。

# 出租车

上海出租车起步价为13元（3公里），超3公里后每公里计价2.4元，超10公里每公里3.6元。等候时间为5分钟算1公里。另加收1元燃油费。夜间（23:00-5:00）收费为白天车费加收 $30 \%$ ， 即夜间起步费为 17元 $+ 7$ 元燃油费 , 超过 3 公里为每公里 3.1 元 , 超过 10 公里为每公里 4.7元 (故夜间出租车以80公里时速于高架路上行驶时计价器每8秒钟增加1 元 )。

郊区出租车（沪 C 牌照，只能在外环线外运营）起步价为 11 元，超出3 公里后每公里 2.4 元，超出 10 公里后不加价格，夜间价格与白天相同。

# 出租车小贴士

在高峰时段打车比较困难，建议打电话提前预约，或者到附近的高级写字楼和酒店门口等。这些场所客流量大，门童发现等车的人比较多也会主动打电话叫车，比在街边拦车保险。

有的酒店门童会挡住外来者以保证本酒店的客人能尽快乘车，最好先进去转一下再出来排队。

夜间打车如果距离比较远或目的地是繁华地区可以要求打 8 折或按日间标准收费，但需要上车前和司机谈好。

慎防克隆出租车，不仅会绕远多收费还可能对人身财产安全造成隐患。识别方法：1) 假出租玻璃多为贴过膜的；2) 假出租车不能刷交通卡；3) 司机会通过客人对目的地的熟悉程度判断乘客是否是外地游客，从而想尽办法绕远获得非法利益。

《半生缘》：张爱玲同名小说拍摄的电影，顾曼璐和顾曼桢是一对姐妹，从小父亲早亡，曼桢由姐姐做舞女抚养长大，后来，曼桢结识沈世钧，两人逐渐情深意笃，但在姐姐和世钧妈妈的干预下，俩人最终只能留下遗憾。这部剧中，几个平凡的众生男女，世钧、曼桢、叔惠、翠芝，把那一点点并不离奇的痴爱怨情，缠来绞去地在一张翻不出去的网里。演了那么多年，也就不年轻了。

# 推荐书籍

《长恨歌》：上海的小家碧玉、“三小姐” 王琦瑶，40 年的情与爱，被一枝细腻而绚烂的笔写得哀婉动人，其中交织着上海这所大都市从四十年代到九十年代沧海桑田的变迁。生活在上海弄堂里的女人沉垒了无数理想、幻灭、躁动和怨望，她们对情与爱的追求，她们的成败，在我们眼前依次展开。王安忆看似平淡却幽默冷峻的笔调，在对细小琐碎的生活细节的津津乐道中，展现时代变迁中的人和城市，被誉为“现代上海史诗”。

《小时代》：郭敬明的笔下，现代的上海，青春逼人。

# 推荐音乐

《上海滩》：这首歌虽是电视剧《上海滩》的主题曲，但是却作为经典传唱不息。整首歌铿锵有力，完美契合了电视剧民国时期上海帮会内的爱恨情仇。

《霞飞路 87 号》：整首歌风格丰富，将沪语和国语的 Rap 配合在一起，副歌和说唱配合和谐，这首歌曲甚至超越了黑棒组合本身的知名度。歌曲 MV 中，黑棒拿着纸扇，在上海潮湿杂乱的弄堂里拗着北方相声演员的造型，唱出的却是活泼嘻哈的歌曲，把两种文化和艺术形态融合得极富冲突，却也极富趣味和美感。
